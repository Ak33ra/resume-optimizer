# Awards, Honors & Competitions

> Copy to `AWARDS.md` and fill in. This section punches above its weight —
> competition results are often the single strongest signal for quant and a
> strong differentiator everywhere. Include scope/ranking and dates so the agent
> can convey how selective each is.

## Competitive programming
- Codeforces — rating / rank (e.g., Candidate Master, 1900) — handle
- ICPC — placement (World Finalist / Regionals rank) — year
- LeetCode / other contest results
- Google Code Jam / Kick Start / Meta Hacker Cup — placement

## Math & science olympiads / exams
- IMO / IOI / national olympiad — medal/rank — year
- Putnam — score/rank — year
- AMC / AIME — score — year

## Data science / ML
- Kaggle — tier (Expert / Master / Grandmaster) — notable competition placements
- Other datathons / ML competitions — placement

## Trading / quant games (for quant_trading)
- Zetamac / mental-math — best score
- Optiver 80-in-8, trading-game or market-making competition results
- Poker / chess — rating or notable results

## Academic & other
- Scholarships, fellowships, grants — name, selectivity, year
- Hackathon wins — event, placement, what you built
- Dean's List / honor societies (if not already in `EDUCATION.md`)
- Any distinction that shows selectivity or excellence
